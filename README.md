<h2><b>Employee Room Assignment System</b></h2>

A simple full-stack web application to manage employees and room assignments.

---

🚀 <b>Tech Stack</b>

Backend

- Python
- FastAPI
- Uvicorn
- Pydantic

Frontend

- React (Vite)
---

⚙️<b> Backend Setup </b>

1. Navigate to backend folder:

cd backend

2. Install dependencies:

pip install -r requirements.txt

3. Run the server:

uvicorn app.main:app --reload

4. Open API docs:

http://127.0.0.1:8000/docs

---

💻 <b>Frontend Setup</b>

1. Navigate to frontend folder:

cd frontend

2. Install dependencies:

npm install

3. Run the development server:

npm run dev

4. Open in browser:

http://localhost:5173

---

✨ <b>Features</b>

- Add Employees
- Add Rooms
- View Employees List
- View Rooms List
- Assign Employees to Rooms
- Prevent duplicate room assignments

---

🧠 <b>Notes</b>

- Backend uses in-memory storage (data resets on server restart)
- Ensure backend is running before using frontend
- API base URL: "http://127.0.0.1:8000"
